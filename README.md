# 🏥 Gyn Appointment Management System

A modern web-based Appointment Management System built with Django. This application helps clinics, hospitals, and healthcare professionals manage patient appointments efficiently through a user-friendly dashboard and complete CRUD operations.

## 📌 Project Overview

The Gyn Appointment Management System is designed to simplify appointment scheduling and management. It allows administrators to create, view, update, search, and delete appointments while monitoring appointment statuses through an interactive dashboard.

## ✨ Features

### 📊 Dashboard

* Total Bookings Counter
* Today's Bookings Summary
* Confirmed Appointments Count
* Pending Appointments Count

### 📅 Appointment Management

* Create New Appointment
* View Appointment Records
* Update Appointment Details
* Delete Appointments
* Search Appointments by Booking ID or Patient Name

### 👩‍⚕️ Patient Information

* Patient Name
* Email Address
* Appointment Date
* Appointment Time
* Appointment Status

### 📌 Status Tracking

* Pending
* Confirmed
* Cancelled
* Completed
